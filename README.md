# phd-prep-summer-2026

Summer preparation ahead of my PhD in the UCL
Department of Computer Science (Oct 2026 – Sep 2030), supervised
by Dr He Wang.

**Theme:** compression × adversarial robustness × safety for AI models on embedded hardware. 
This repo holds the code I build while working in this period — PyTorch foundations, adversarial attacks, model compression, transformers/LLMs, and on-device deployment.

## Structure
Organised by week, matching the sprint timeline:
- 'week1/' - PyTorch fundamentals: tensors, autograd, a tiny MLP
- 'week2/' - CNN on CIFAR-10 (full training pipeline)
- ... (added as each week is reached)

## Running the code
Notebooks are written for Google Colab (free tier, T4 GPU).
Primary framework: PyTorch. Open any notebook in Colab and run top-to-bottom.

In progress — updated with daily commits.
